# 二手平台管理系统 java1108

## 项目概述

二手平台管理系统是一款基于 Spring Boot + Vue 前后端分离架构的开源管理系统。它集成了二手商品发布、管理、交易等功能，适用于构建中小型二手交易平台。

## 技术栈

### 前端技术

- Vue
- Element-UI

### 后端技术

- Spring Boot
- MyBatis Plus
- MySQL

## 功能模块

系统包含但不限于以下功能模块：

- 用户管理
- 商品管理
- 交易管理
- 订单管理
- 数据统计

## 系统角色

- 系统管理员
- 普通用户
- 商家用户

## 运行环境

- Java 1.8+
- MySQL 5.7+
- Node.js 12+
